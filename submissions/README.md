# Submissions

Team submissions land here (one folder per team, one `.py` file inside).

See [`../SUBMITTING.md`](../SUBMITTING.md) for instructions.

## Example structure

```
submissions/
├── alpha_centauri/
│   └── submission.py
├── voyager/
│   └── submission.py
└── yuri_gagarin_fan_club/
    └── submission.py
```
